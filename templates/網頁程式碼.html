語法解釋


<html></html>

此標籤是置於整個文件的開始和結束之處，以供瀏覽器識別此文件為合法的文件。

<head></head>

此標籤是用以表示標頭資訊的開始與結束，在<head>與</head>之間可加入其它標籤<title></title>或<meta>等。

<meta>

標籤位於文檔的頭部，不包含任何內容。其屬性定義了與文檔相關聯的名稱/值對。

<title></title>

此標籤中的就是此網頁的標題，也就是您瀏覽器最左上面的標題，若沒設則只會顯示成此網頁所在的網址。

<body></body>

此標籤中所寫的內容即會顯示於網頁裡面。

<section></section>

定義一個大的段落。

<article></article>

定義段落裡的一篇文章。

<aside></aside>

網頁內容以外的項目，大多都是側邊攔目。

<figure></figure>

定義內體內容的區塊。

<figcaption></figcaption>

定義媒體區塊的標題文字。

<header></header>

定義段落中的標題，不是<head></head>喔。

<footer></footer>

定義頁尾的資訊。

<nav></nav>

定義導覽列的內容。

<br>

此標籤是跳下一行，若您的本文沒使用段行標籤的話，基本上則會寫滿一個畫面才會跳下一行，而使用<br>的話則可在適當的地方讓文章能換行。

<p></p>

此標籤是段落，若您的本文沒使用段行標籤的話，基本上則會寫滿一個畫面才會跳下一行，而使用<p>的話則可自動跳到下一段落，也就是中間會空一行，這與<br>有不同之處。

<hr>

此標籤會產生一條分格線。

<hr size=4>

可使用size來控制分格線的高度。

<hr width=500>

可使用width來控制分格線的長度。

<hr color=ff0000>

可使用color來控制分格線的顏色(請參照色碼表)本範例為紅色。

<hr align=right>

可使用align來控制分格線的位置(預設為置中center)右right、左left。

<center></center>

將所要置中的文字或圖片置於<center>與</center>之間即可。

<strong></strong>

用以加強文字的粗體效果。

<!---->

註解之意，於<!---->之間的所有文字皆不會出現。

<font size=""></font>

只要將要放大或要縮小的文字放置於語法<font size="">與</font>之間。

<font color=""></font>

改變字的顏色。

<font face="字型全名"></font>

改變字型。

<b></b>

於標籤內的文字變為粗體。

<ul></ul>

非排序性列表。

<li></li>

非排序性項目符號。

<marquee>要出現的字或圖</marquee>

跑馬燈（左進右出式）。

<marquee behavior=alternate></marquee>

跑馬燈（左右來回式）。

<a href="連結網址">說明文字</a>

用以連結網址，而說明文字是讓人知道這連結是連到什麼的。

<a href="mailto:你的E-Mail">說明文字</a>

電子郵件(E-Mail)超連結。

<a href="#書籤"></a>

書籤的超連結，只要在頁內做好各點的書籤就可以自由活動了。很常用在回到頂端TOP的地方。

<img src="圖檔的所在" alt="圖片註解">

貼圖標籤及圖片註解。

<table></table>

此標籤是表格的基本，而其基本設定還有很多。

<tr></tr>

此標籤是列之意，而每列中可有很多的儲存格。

<td></td>

此標籤是屬一般的資料儲存格。

border=設定[表格]

此標籤是設定外框的厚度大小，例如border=3，若0則沒有外框。

text="設定"

設定基本文字的顏色，例如text="0000ff"，內定是黑色。

link="設定"

設定超連結文字的顏色，例如link="ff0000"，內定是藍色。

bgcolor="設定"

設定背景顏色，例如bgcolor="ffcc00"，內定是白色。

background

設定背景圖片，用法background="圖片所在網址"。

<div></div>

其間包起來的部份就是區塊內容，可以是文字、圖片、影片、框架 ... 等網頁內容，開頭的<div>區塊內可以設置 id、class 或直接開始一段樣式設計，也就是 style 的部份。

id

它的作用是在頁面的所有元素中，將某個元素獨立辨識出來。

class

有時候想做的並非獨立識別文件中的單一元素，而是將幾個元素從其他元素中獨立出來。

style

用於為HTML 文檔定義樣式信息。

<script></script>

標籤用於定義客戶端腳本，比如JavaScript。

<noscript></noscript>

元素用來定義在腳本未被執行時的替代內容（文本）。 此標籤可被用於可識別<script></script>標籤但無法支持其中的腳本的瀏覽器。

<input type=text>

文字欄位。

<textarea></teatarea>

文字輸入方塊。

<input type=submit>

執行按鈕，執行由 action 後所指定的 cgi 程式。

<input type=reset>

取消按鈕，還原表單內的資料為內定值。

<input type=button>

一般按鈕，使用者自行設計的按鈕。
